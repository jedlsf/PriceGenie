    Interface PricingModel

    Represents the pricing model for a specific unit of measurement. This model defines how pricing is calculated based on unit type, unit price, quantity, and base price.

    Defines the pricing type. For example:

    • PER UNIT - Pricing based on the quantity of units multiplied by the unit price.
